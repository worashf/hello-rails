# hello-rails

This project is created to learn how to create rails project with PostgreSQL database setup.

# Built with

1. Rails 7.0.4
2. Ruby 3.1.2
3. PostgreSQL

# Setup and run

1.  clone https://github.com/worashf/hello-rails
2.  cd hello-rails
3.  run 'rails server'
4.  check on browser 'http://127.0.0.1:3000/
